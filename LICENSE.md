## 🚨 Common Issues & Fixes

### ❌ Menu doesn't open after injection
**Fix:** Run `NorthInjector.exe` as Administrator. Ensure GTA 5 is fully loaded to the main menu before injecting — not the loading screen.

### ❌ Antivirus deletes the injector
**Fix:** Add the North folder to your antivirus exclusion list. This is a false positive caused by injection behavior, not malware.

### ❌ Game crashes on launch with menu loaded
**Fix:** Verify your GTA 5 game files through Steam/Rockstar Launcher, then re-inject. A corrupted game file is usually the culprit.

### ❌ Features are greyed out in Online mode
**Fix:** Some features are intentionally disabled in Online for your protection. Switch to a private session to unlock full functionality.

### ❌ Injector says "Process not found"
**Fix:** Make sure GTA 5 is running before launching the injector. Also check that you're running the correct version (Steam vs. Epic vs. Rockstar).

### ❌ Auto-farm script stops mid-session
**Fix:** This is usually triggered by a session event (e.g., another player joining). Re-activate the script from the menu or switch to a private session.

### ❌ Menu freezes / low FPS with overlay active
**Fix:** Lower your in-game resolution or disable post-processing effects. The overlay adds GPU overhead — ensure you meet the recommended system requirements.
